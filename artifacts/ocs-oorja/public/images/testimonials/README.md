# Testimonial photos

Put real customer photos here and reference them from
`src/data/testimonials.ts` via the `photo` field.

- **Naming:** lowercase, hyphenated full name — e.g. `ravi-kumar.jpg`
- **Format:** square crop works best (it is shown as a circle), JPG or PNG,
  at least 200×200 px.
- **Reference from data:** `photo: "/images/testimonials/ravi-kumar.jpg"`
- If a testimonial has no `photo` (or the file is missing), the site
  automatically shows a clean initials avatar instead — nothing breaks.

Only use photos of the actual customer, with their permission. Never use
stock photos of strangers as customer photos.

# Downloadable documents

Two kinds of documents live here:

- `brochures/` — marketing brochures (family overviews or multi-product)
- `datasheets/` — technical datasheets (one product/model each)

## File naming

Use lowercase, hyphen-separated names:

```
brochures/<family-or-topic>-brochure.pdf
datasheets/<family>-<model>-datasheet.pdf
```

### Examples

```
brochures/lifepo4-batteries-brochure.pdf
datasheets/lifepo4-batteries-12v8-100ah-datasheet.pdf
datasheets/ac-ev-chargers-7kw-wallbox-datasheet.pdf
```

## Rules

- **PDFs only.**
- **Never overwrite** an existing file with different content. Version it instead,
  e.g. `lifepo4-batteries-12v8-100ah-datasheet-v2.pdf`.
- Compress large PDFs before uploading.

> The product catalog references these paths from a single data source, so a
> correctly named file becomes a working download link automatically.

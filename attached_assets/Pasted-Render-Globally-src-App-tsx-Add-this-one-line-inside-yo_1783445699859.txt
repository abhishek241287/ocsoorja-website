Render Globally: src/App.tsx
Add this one line inside your app's root div, outside the router so it persists across pages:
tsx
// src/App.tsx
import { WhatsAppWidget } from "@/components/global/WhatsAppWidget";

function App() {
  return (
    <div className="min-h-screen bg-background text-foreground">
      {/* ... your existing router/layout ... */}
      
      <WhatsAppWidget />
    </div>
  );
}

One-Line Fix if Pulse + Expansion Conflicts
If the pulse animation makes the pill expansion feel janky, replace the pulse with a subtle shadow breathe (no scale) on the container, and keep the icon static:
css
@keyframes wa-glow {
  0%, 100% { box-shadow: 0 4px 20px rgba(37, 211, 102, 0.22); }
  50% { box-shadow: 0 6px 28px rgba(37, 211, 102, 0.32); }
}
Use wa-glow instead of wa-pulse. No scale = no layout shift = smooth expansion
